# Contributing

TrainDeck is proprietary software maintained by Maple Vibe Inc.

Issues, feature requests, bug reports, compatibility notes, and locomotive
profile observations are welcome.

Pull requests, patches, source-code contributions, submitted profile files, and
other copyrightable contributions are not accepted unless Maple Vibe Inc. has
explicitly requested them and a separate written contribution agreement is in
place.

If you want to help, the best path is to open an issue with:

- Train Sim World version
- Route and locomotive name
- Screenshot of the locomotive info panel
- What worked and what did not
- Any cab snapshot or bridge log details requested by Maple Vibe Inc.
